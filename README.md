# ImageViewer
Image viewing program with speed and color accuracy as main goal


## Roadmap
as you can clearly this is very early stages of this project but currently goals are following:
 * Prepare something that runs fast
 * Bit switching to get one extra virtual color depth
 * High color resolution rendering: 10bpc (30bit), 12bpc (36) rendering support for systems that supports these modes
 * Color profiles support
 * Advanced editing mode
 * Precompiled plugins
 * Scripted plugins
 * Compiling plugins
